Demos


Shaku is a pure JavaScript framework for web games development that emphasize simplicity, flexability and freedom.
If you don't know it, read more about it here.

The following is a set of small demos to illustrate different features of Shaku lib.

Basic

  • Hello World - how to init Shaku and create a main loop.
  • Sandbox - a sandbox application to play around with Shaku.

Gfx

  • Draw - illustrate the most basic draw() method.
  • Colors - show the built-in color consts.
  • Sprites - drawing using sprites.
  • Sprites Group - grouping sprites together.
  • Blend Modes - illustrate the supported rendering blend modes.
  • Drawing Text - illustrate how to render text using font textures.
  • Drawing Lines - illustrate how to render lines.
  • Custom Effects - basic example on how to create custom effects (shaders).
  • Fullscreen - show how to stretch canvas on the entirely of the screen.
  • Render Target - show how to use render targets.
  • Wrap & Filter modes - show texture wrap and filter modes.

Sfx

  • Basic Audio - sound effects and ambience.
  • Audio Mixer - a utility to mix sound effects and music.

Input

  • Basic Input - keyboard and mouse input.

Collision

  • Basic Collision - illustrate basic collision detection between shapes.

Performance

  • Simple Performance Test - a performance test with and without batching, using a single texture.
  • Harder Performance Test - a performance test with and without batching, using multiple textures.

Miscs

  • Animators - using animators to animate sprites.
  • Json Asset - loading json assets example.
  • Binary Asset - loading binary assets example.

Games

  • Snake - a snake game example.
  • Fishy - eat smaller fish to grow, dodge bigger fish to survive.
  • Tilemap & Camera - a simple tilemap editor with moving camera.
