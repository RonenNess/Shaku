Demos


Shaku is a pure JavaScript framework for web games development that emphasize simplicity, flexability and freedom.
If you don't know it, read more about it here.

The following is a set of small demos to demonstrate different features of Shaku lib.

Basic

  • Hello World - how to init Shaku and create a main loop.
  • Sandbox - a sandbox application to play around with Shaku.

Gfx

  • Draw - demonstrate the most basic draw() method.
  • Colors - show the built-in color consts.
  • Sprites I - drawing sprites.
  • Sprites II - drawing sprites with different properties.
  • Vertices - drawing sprites with vertices.
  • Sprites Group - grouping sprites together.
  • Blend Modes - demonstrate the supported rendering blend modes.
  • Drawing Text - demonstrate how to render text using font textures.
  • Drawing Lines - demonstrate how to render lines.
  • Custom Effects I - example on how to create custom effects (shaders).
  • Custom Effects II - example on how to create custom effects (shaders) with two textures.
  • Fullscreen - show how to stretch canvas on the entirely of the screen.
  • Render Target - show how to use render targets.
  • Wrap & Filter modes - show texture wrap and filter modes.
  • Batching - basic sprite batching test.

Sfx

  • Basic Audio - sound effects and ambience.
  • Audio Mixer - a utility to mix sound effects and music.

Input

  • Basic Input - keyboard and mouse input.

Collision

  • Basic Collision - demonstrate basic collision detection between shapes.
  • Collision Flags - demonstrate collision detection with collision flags.
  • Collision Tilemap - demonstrate the collision tilemap shape.

Performance

  • Simple Performance Test - a performance test with a single texture.
  • Harder Performance Test - a performance test with and without batching, using multiple textures.
  • Static Performance Test - a performance test with static sprites.

Miscs

  • Vectors - Vectors and vectors math.
  • Animators - using animators to animate sprites.
  • Json Asset - loading json assets example.
  • Binary Asset - loading binary assets example.
  • Perlin Noise Generator - using the built-in perlin noise generator.
  • Path Finder - using the built-in path finding utility.

Games

  • Snake - a snake game example.
  • Fishy - eat smaller fish to grow, dodge bigger fish to survive.
  • Tilemap & Camera - a simple tilemap editor with moving camera.
